Google Book Search

A simple book search engine from Google's database, made for the purpose of facilitating the verification of price, availability and general data from existing books.
The app contains a Print Type filter to select between Books and Magazines, or both, and a Book Type filter to select between: Previewable, Full-Text, E-Books, Free E-Books and Paid E-Books, or all. The data returned includes a plethora of categories, but the only ones used are: book image (smaller size), book name, authors, price, availability and description.
